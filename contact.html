Notable Projects:

  1. SimpleKitchen

    SimpleKitchen is intended to be your all-in-one kitchen assistant, all for free. Recipe planning & storage, inventory management and automatically finding recipes compliant with your diet.

  2. NEA-Project

    This was the project I worked on for Computer Science A-Level. I used C to write a graphical application for Casio calculators that would allow users to solve optimisation problems using the Simplex algorithm. I was inspired to take on this project as these problems were part of my Futher Mathematics course.

  3. JokeAPI-Python

    JokeAPI-Python is a Python package that wraps around JokeAPI to provide developers with an easy way to interact with the API. I've developed and maintained this package since Feb 2020, almost 4 years at this point, and it has garnered some reasonable usage, given that it is a fairly well-serviced niche already.

  4. foldingathome

    foldingathome is a Python package that can be used to fetch statistics from Folding@Home's statistics API. I first developed it in 2019, and recently brought it up to date with Folding@Home's new API. Along with this update, I practically did a total rewrite, significantly improving the quality of code.
